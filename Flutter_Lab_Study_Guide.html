🧪 Experiment 1 – SafeArea Widget

What is SafeArea? SafeArea adds padding so your content doesn't go behind the phone's status bar, notch, or home button area. Think of it as a "safe zone" for your UI. 
import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(home: MyApp()));
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: SafeArea(
        child: Text(
          'This text is inside SafeArea!',
          style: TextStyle(fontSize: 22),
        ),
      ),
    );
  }
}
💡 Tip: Always wrap your main content in SafeArea to avoid UI elements hiding behind the status bar or notch.

🧪 Experiment 16 – DropdownButton Widget

DropdownButton shows a list of options when clicked, and the user selects one. Requires StatefulWidget because the selected value changes. 
import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(home: MyApp()));
}

class MyApp extends StatefulWidget {
  @override
  _MyAppState createState() => _MyAppState();
}

class _MyAppState extends State {
  String selectedValue = 'Apple';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Dropdown Button')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('Selected: $selectedValue', style: TextStyle(fontSize: 22)),
            SizedBox(height: 20),
            DropdownButton(
              value: selectedValue,
              items: ['Apple', 'Banana', 'Mango', 'Orange']
                  .map((fruit) => DropdownMenuItem(value: fruit, child: Text(fruit)))
                  .toList(),
              onChanged: (newValue) {
                setState(() {
                  selectedValue = newValue!;
                });
              },
            ),
          ],
        ),
      ),
    );
  }
}
💡 Key Concept: Dropdown needs StatefulWidget + setState() to update the selected value on screen.

🧪 Experiment 20 – GridView Widget

GridView displays items in a 2D grid (rows and columns). Like a photo gallery layout. 
import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(home: MyApp()));
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('GridView')),
      body: GridView.count(
        crossAxisCount: 2,       // 2 columns
        crossAxisSpacing: 10,
        mainAxisSpacing: 10,
        padding: EdgeInsets.all(10),
        children: List.generate(8, (index) {
          return Container(
            decoration: BoxDecoration(
              color: Colors.primaries[index % Colors.primaries.length],
              borderRadius: BorderRadius.circular(12),
            ),
            child: Center(
              child: Text(
                'Item ${index + 1}',
                style: TextStyle(color: Colors.white, fontSize: 20),
              ),
            ),
          );
        }),
      ),
    );
  }
}
💡 Tip: crossAxisCount: 2 means 2 columns. Change it to 3 for 3 columns. List.generate(8, ...) creates 8 items.

🧪 Experiment 21 – ListView Widget

ListView shows a scrollable list of items, like a WhatsApp chat list or a contacts list. 
import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(home: MyApp()));
}

class MyApp extends StatelessWidget {
  final List<String> fruits = ['Apple', 'Banana', 'Mango', 'Orange', 'Grapes', 'Pineapple'];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('ListView')),
      body: ListView.builder(
        itemCount: fruits.length,
        itemBuilder: (context, index) {
          return ListTile(
            leading: Icon(Icons.local_florist, color: Colors.green),
            title: Text(fruits[index], style: TextStyle(fontSize: 18)),
            trailing: Icon(Icons.arrow_forward_ios),
          );
        },
      ),
    );
  }
}
💡 Tip: Use ListView.builder for large lists — it only builds items visible on screen (efficient). ListView(children: []) builds all at once.

🧪 Experiment 27 – Navigation and Routing

Navigation = moving between screens. Navigator.push() goes to next screen. Navigator.pop() goes back. 
import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(home: HomeScreen()));
}

// Screen 1 - Home
class HomeScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Home Screen')),
      body: Center(
        child: ElevatedButton(
          child: Text('Go to Second Screen', style: TextStyle(fontSize: 18)),
          onPressed: () {
            Navigator.push(
              context,
              MaterialPageRoute(builder: (context) => SecondScreen()),
            );
          },
        ),
      ),
    );
  }
}

// Screen 2 - Second
class SecondScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Second Screen')),
      body: Center(
        child: ElevatedButton(
          child: Text('Go Back', style: TextStyle(fontSize: 18)),
          onPressed: () {
            Navigator.pop(context);
          },
        ),
      ),
    );
  }
}
💡 Key: Navigator.push() = go forward. Navigator.pop() = go back. MaterialPageRoute wraps the destination screen.

🧪 Experiment 29 – Gesture Detection

GestureDetector detects touch events: tap, double tap, long press, swipe (drag), etc. 
import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(home: MyApp()));
}

class MyApp extends StatefulWidget {
  @override
  _MyAppState createState() => _MyAppState();
}

class _MyAppState extends State {
  String gestureText = 'Touch the box!';
  Color boxColor = Colors.blue;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Gesture Detector')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text(gestureText, style: TextStyle(fontSize: 22, fontWeight: FontWeight.bold)),
            SizedBox(height: 30),
            GestureDetector(
              onTap: () {
                setState(() {
                  gestureText = 'Single Tap!';
                  boxColor = Colors.green;
                });
              },
              onDoubleTap: () {
                setState(() {
                  gestureText = 'Double Tap!';
                  boxColor = Colors.red;
                });
              },
              onLongPress: () {
                setState(() {
                  gestureText = 'Long Press!';
                  boxColor = Colors.purple;
                });
              },
              child: Container(
                height: 150,
                width: 250,
                color: boxColor,
                alignment: Alignment.center,
                child: Text(
                  'Touch Me!',
                  style: TextStyle(color: Colors.white, fontSize: 24, fontWeight: FontWeight.bold),
                ),
              ),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                setState(() {
                  gestureText = 'Touch the box!';
                  boxColor = Colors.blue;
                });
              },
              child: Text('Reset'),
            ),
          ],
        ),
      ),
    );
  }
}
💡 Common gestures: onTap, onDoubleTap, onLongPress, onHorizontalDragUpdate, onVerticalDragUpdate.
